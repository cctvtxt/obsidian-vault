HTTP/3 (2022) идёт дальше — заменяет TCP на протокол QUIC (работает поверх UDP). Ключевые преимущества:

- устранение head‑of‑line blocking на уровне пакетов;
- ускоренное установление соединения (0‑RTT);
- встроенная шифрование (QUIC включает TLS 1.3);
- лучшая работа при потере пакетов и переключении сетей.
