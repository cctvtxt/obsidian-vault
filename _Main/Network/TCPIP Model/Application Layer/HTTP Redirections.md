[[HTTP 1.1]]

**Перенаправление URL** это способ присвоения нескольких URL-адресов странице, форме, всему веб-сайту или веб-приложению. Для этой операции в протоколе HTTP предусмотрен специальный тип ответа - HTTP-перенаправление.

Перенаправления служат множеству целей: Временные перенаправления во время технического обслуживания сайта или его простоя, постоянные перенаправления для сохранения существующих ссылок/закладок после изменения URL-адресов сайта, промежуточных страниц при загрузке файлов и т. д.

В протоколе HTTP перенаправление инициируется сервером, отправляющим специальный ответ **redirect** на запрос. Ответы с перенаправлением имеют коды состояния, **начинающиеся с 3**, и заголовок **Location**, содержащий URL для перенаправления.

Когда браузер получает редирект, он сразу же загружает новый URL, указанный в заголовке **Location**. Пользователи редко замечают перенаправление
