[[HTTP Status Codes]]

##### 3* класс кодов (перенаправление на другой адрес). Эти коды информируют, что для достижения успешной операции нужно будет сделать другой запрос, возможно, по другому URL.

300 Multiple Choices: указывает на несколько возможных вариантов ресурса. Клиент может выбрать один из них. Сервер предоставляет список вариантов в теле ответа.

301 Moved Permanently: ресурс был окончательно перемещен на новый URL. Важно для SEO, так как передает "вес" страницы новому адресу.

302 Found: ресурс временно находится по другому URL. Клиент должен использовать старый URL для будущих запросов. В отличие от 301, не передает SEO-вес.

303 See Other: указывает, что ответ на запрос можно найти по другому URL с помощью метода GET. Используется после POST-запроса, чтобы избежать повторной отправки формы.

304 Not Modified: ресурс не изменился с момента последнего запроса. Экономит трафик, так как сервер не отправляет тело ответа.

305 Use Proxy: запрашиваемый ресурс должен быть доступен через прокси. URL прокси указывается в заголовке `Location`.

306 Unused: зарезервирован, но больше не используется.

307 Temporary Redirect: ресурс временно перемещен, но, в отличие от 302, требует, чтобы клиент не менял метод запроса (например, POST остается POST).

308 Permanent Redirect: ресурс окончательно перемещен, и, как и 307, требует сохранения исходного метода запроса. Аналог 301, но с сохранением метода.
