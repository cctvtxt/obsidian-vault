### 5. Ключи API

**Суть:** уникальные строки, генерируемые для доступа к API. Передаются в заголовках, параметрах запроса или теле запроса.

**Плюсы:**

- простота идентификации потребителей API;
- возможность отзыва и перевыпуска токенов.

**Минусы:**

- риск перехвата ключей;
- ограниченная поддержка MFA.

**Когда использовать:** веб‑приложения, ориентированные на потребителя, публичные API.
