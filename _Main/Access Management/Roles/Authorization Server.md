[[Authentification]]

Authorization Server (Сервер Авторизации) отвечает за проверку личности владельца ресурса и получение его согласия (consent) на предоставление доступа приложению-клиенту. В случае успеха он выпускает токен доступа (access token).

Это центральный компонент в фреймворке OAuth 2.0. В протоколе OpenID Connect (OIDC) его роль выполняет OpenID Provider (OP), который расширяет его функциональность для аутентификации.
