[[JSON Web Token]]

Стандартный JWT (RFC 7519) использует JWS (header.payload.signature), где payload только Base64-кодирован и подписан, но читаем для всех



![[Pasted image 20260308170022.png]]





Хранение секретного ключа или HMAC-секрета:

**Клиент** обладает только JWT-токеном, ему не передается секретный ключ или HMAC secret.

**Сервер** хранит ключ (RSA) или HMAC secret в переменных окружения / хранилище секретов.
