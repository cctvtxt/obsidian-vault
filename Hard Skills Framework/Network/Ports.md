Порты нужны для разграничения подключений к серверу, для работы с конкретными программами на сервере: например, веб-сервер, ssh-сервер.

**Количество программных портов** ограничено с учётом 16-битной адресации — **65 535**. Все порты разделены на три диапазона:

- общеизвестные (или системные, 0–1023);
- зарегистрированные (или пользовательские, 1024–49151);
- динамические (или частные, 49152–65535).
