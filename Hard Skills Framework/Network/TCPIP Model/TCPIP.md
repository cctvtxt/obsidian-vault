[[Protocol Stacks]]

**TCP/IP** — практичная модель с четырьмя уровнями, охватывающая функции OSI компактно.

- **Сетевой доступ (Network Interface)**: физическая/канальная передача (кабели, оптоволокно, Wi-Fi), адресация в LAN (**MAC-адреса**).
- **Интернет (Internet)**: маршрутизация пакетов (**IP** через AS).
- **Транспортный (Transport)**: надёжность, последовательность (**TCP**, **UDP**).
- **Прикладной (Application)**: протоколы (**HTTP/HTTPS**, **FTP**, **SMTP**, **DNS**) для пользовательских сервисов.

Модель ориентирована на реальные задачи: доставку, порядок, целостность пакетов и взаимодействие приложений.
