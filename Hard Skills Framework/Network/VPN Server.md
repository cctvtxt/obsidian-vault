Virtual Private Network переводится как виртуальная частная сеть.

Впервые термин VPN придумала организация Microsoft. Управляющие компании хотели, чтобы сотрудники имели доступ к рабочей сети со своих домашних компьютеров. Однако возникла проблема: корпоративная информация должна была оставаться закрытой для сторонних лиц. Решение этой проблемы стало возможным благодаря созданию виртуальной частной сети — VPN. Проще говоря, ВПН — это дополнительная сеть, которая осуществляет соединение с необходимым сервером по специальному защищенному каналу (туннелю). Передаваемые по такому каналу данные защищаются шифрованием, а IP-адрес пользователя подменяется на общий IP. Таким образом, человека, который использует ВПН, нельзя идентифицировать. Даже интернет-провайдер не знает, на какие ресурсы заходит пользователей, какие покупки он совершает и т. д.

**Но в современном мире** термин VPN в большинстве случаев стал **синонимом прокси-подключения** к интернету. То есть идея VPN используется для того, чтобы подключиться к серверу, который находится например в Польше, однако полноценный VPN
