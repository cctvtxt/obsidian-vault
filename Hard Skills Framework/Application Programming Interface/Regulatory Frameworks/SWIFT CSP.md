[[API Compliance]]

SWIFT Customer Security Programme (CSP) — это программа, запущенная SWIFT для повышения уровня кибербезопасности в глобальной банковской экосистеме. Она устанавливает набор обязательных и рекомендуемых мер контроля для всех финансовых учреждений, подключенных к сети SWIFT.

Программа решает задачу защиты от кибератак, направленных на компрометацию локальной инфраструктуры банков и отправку мошеннических платежных поручений через SWIFT.

CSP базируется на наборе контролей, описанных в документе Customer Security Controls Framework (CSCF). Эти контроли охватывают три ключевые области: защита локальной среды, ограничение доступа и управление взаимоотношениями, а также обнаружение аномалий и реагирование на инциденты.

# Вводимые ограничения

API, интегрированные с системами SWIFT, должны быть развернуты в защищенном и изолированном сегменте сети. Доступ к этим API должен быть строго ограничен, а все коммуникации — зашифрованы.

Требуется внедрение **многофакторной аутентификации (MFA)** для всех пользователей, имеющих доступ к системам, связанным со SWIFT, включая операторов, работающих через API.

Система должна обеспечивать непрерывный мониторинг транзакций и поведения пользователей для выявления аномалий. API должен быть интегрирован с системами обнаружения вторжений (IDS) и SIEM для своевременного реагирования на подозрительную активность. Обязательно регулярное проведение тестов на проникновение.

# Практический путь внедрения

Соответствие SWIFT CSP является обязательным условием для всех участников сети SWIFT. Ежегодно каждая организация должна проходить самоаттестацию (а для некоторых — независимый аудит) на соответствие контролем CSCF.

Разработка API, взаимодействующего со SWIFT, требует построения высокозащищенной инфраструктуры. Это включает физическую и логическую изоляцию, внедрение строгих политик паролей и управления доступом, а также использование передовых средств защиты. Несмотря на сложность, это единственный способ обеспечить безопасность финансовых операций и сохранить доверие в международной банковской системе.
