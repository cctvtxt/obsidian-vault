[[Direct Auth]]

**Server** — это серверное приложение, которое хранит базу данных учетных записей пользователей, проверяет предоставленные учетные данные и принимает решение об аутентификации.

Server выполняет центральную роль в процессе аутентификации: получает запрос от клиента с учетными данными, сверяет их с хранящейся информацией (хэшами паролей, сертификатами или другими аутентификационными данными), создает сессию или токен при успешной проверке
