[[Federative Auth]]

Resource Server — это сервер, на котором хранятся защищённые ресурсы. Его задача — принимать запросы от клиентских приложений, проверять валидность приложенного к ним токена доступа (access token) и, в случае успеха, возвращать запрошенный ресурс.

Эта роль является неотъемлемой частью архитектуры OAuth 2.0 и систем на базе OpenID Connect, где требуется защита API.
