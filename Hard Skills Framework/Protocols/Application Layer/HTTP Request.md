[[HTTP]]

В Интернете основным протоколом является, протокол передачи гипертекста, более известный под своим сокращением **HTTP**. В веб-браузере, "**http**" указывает браузеру использовать правила **HTTP** при общении с сервером.

Взаимодействие в протоколе **HTTP** строится вокруг концепции, называемой **Циклом Запрос-Ответ** (**the Request-Response Cycle**).

_Чтобы сделать запрос, клиенту необходимо указать четыре вещи:_

1. **_URL_** _(единый указатель ресурсов) (_**_Uniform Resource Locator_**_)_: на самом деле спецификация требует, чтобы запрос содержал **URI** (универсальный идентификатор ресурса) (**Uniform Resource Identifier**), подмножеством которого являются **URL**-адреса** (единые указатели ресурсов), наряду с **URN** (унифицированными именами ресурсов) **(Uniform Resource Name)\*\*

2. **_Список заголовков_** _(_**_Headers_**_) = (List of headers)_
3. **_Тело_** _(_**_Body_**_)_
