[[Monitoring]]

Заголовок запроса `Referer` (исторически неверное написание слова "referrer") содержит URL страницы, с которой был сделан текущий запрос.

# Описание

Когда вы кликаете по ссылке на странице `A`, которая ведет на страницу `B`, браузер при запросе страницы `B` добавит заголовок `Referer: <URL страницы A>`.

`Referer: https://example.com/previous-page.html`

# Юзкейсы

*   **Веб-аналитика**: Позволяет отслеживать источники трафика. Системы аналитики (Google Analytics, Яндекс.Метрика) используют `Referer`, чтобы показать, с каких сайтов, поисковых систем или социальных сетей приходят пользователи.
*   **Логирование и отладка**: Помогает понять путь пользователя по сайту, что может быть полезно при анализе ошибок.
*   **Оптимизация навигации**: На основе данных `Referer` можно улучшать навигацию, делая более доступными страницы, с которых часто приходят пользователи.
*   **Простая защита от "хотлинкинга"**: Некоторые серверы проверяют `Referer` при запросе изображений или других ресурсов, чтобы убедиться, что они запрашиваются с их собственного сайта, а не встроены на чужой.

# Проблемы и особенности

*   **Приватность**: `Referer` может раскрывать конфиденциальную информацию, если она содержится в URL (например, `.../reset-password?token=...`).
*   **`Referrer-Policy`**: Для управления тем, какая информация (и отправляется ли вообще) передается в `Referer`, существует заголовок ответа `Referrer-Policy`. Он позволяет, например, отправлять только домен (`strict-origin-when-cross-origin`) или не отправлять заголовок вовсе (`no-referrer`).
*   **Не отправляется в некоторых случаях**:
    *   При переходе с `HTTPS` на `HTTP` `Referer` не отправляется по соображениям безопасности.
    *   Если пользователь вводит URL вручную или переходит из закладок, заголовок будет отсутствовать.
    *   Настройки браузера или расширения могут блокировать его отправку.
*   **Орфографическая ошибка**: Правильное написание — "referrer", но в стандарте HTTP закрепилась ошибка. В новых API (например, `Referrer-Policy`) используется уже корректное написание.
