[[HTTP Headers]]

Заголовки, связанные с **содержимым**, описывают тело HTTP-сообщения. Они помогают клиенту и серверу правильно интерпретировать данные, которые они передают друг другу, указывая на их тип, размер и кодировку.

1. `Content-Type` — указывает MIME-тип тела.
2. `Content-Length` — размер тела сообщения.
3. `Content-Encoding` — кодировка/сжатие тела, например `gzip`.
4. `Accept` — какие типы ответа клиент готов принять.
5. `Accept-Encoding` — какие схемы сжатия клиент принимает.
6. `Accept-Language` — какие языки предпочитает клиент.
7. `Accept-Charset` — какие наборы символов клиент принимает.
8. `Content-Language` — язык содержимого ответа.
9. `Content-Location` — альтернативный URI ресурса или представления.
10. `Transfer-Encoding` — способ передачи тела, например `chunked`.
