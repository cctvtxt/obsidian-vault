[[HTTP Headers]]

Заголовки **кэширования** управляют тем, как и на какой срок ответы могут сохраняться в кэше браузера или промежуточных прокси-серверов. Эффективное кэширование значительно ускоряет загрузку сайтов и снижает нагрузку на сервер.

1. `Cache-Control` — управляет поведением кэша.
2. `Expires` — задаёт срок устаревания ответа.
3. `ETag` — идентификатор версии ресурса.
4. `If-None-Match` — условный запрос по `ETag` для проверки изменений.
5. `If-Modified-Since` — условный запрос по дате последнего изменения.
6. `Vary` — указывает, какие заголовки влияют на кэш.
7. `Last-Modified` — сообщает дату последнего изменения ресурса.
8. `Pragma` — устаревший заголовок, часто `no-cache`.
9. `Age` — показывает, сколько ответ уже живёт в кэше.
10. `If-Match` — проверяет, совпадает ли текущая версия ресурса.
