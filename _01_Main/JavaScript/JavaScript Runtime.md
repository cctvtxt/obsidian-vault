[[JavaScript]]

**JS Runtime** (или **JavaScript Runtime**) — это среда выполнения JavaScript‑кода. Это окружение, в котором скрипт на JavaScript запускается, интерпретируется и выполняется программно.

## Что такое JS Runtime?

- JS Runtime — это программная среда, которая содержит движок JavaScript (например, V8, SpiderMonkey), event loop, встроенные API и другие компоненты, необходимые для работы кода. 
- Без runtime чистый JavaScript‑файл не может «запуститься» сам по себе: ему нужен хост (браузер, Node.js, Deno и т.п.), который и выступает в роли JS Runtime.  

## Основные компоненты JS Runtime

- **JavaScript‑движок** — разбирает и исполняет код, включая JIT‑компиляцию. 
- **Event loop** — управляет асинхронными операциями и событиями, обеспечивая неблокирующую модель выполнения. 
- **Хост‑API** — предоставляет доступ к системным возможностям (файловая система, сеть, таймеры и т.д.), которые не описаны в спецификации ECMAScript. 

## Примеры JS Runtime

- **Браузер** — классический вариант JS Runtime: веб‑страница, движок браузера и встроенные Web‑API (DOM, `fetch`, `setTimeout` и др.). 
- **Node.js** — серверный JS Runtime, построенный на V8, который позволяет запускать JavaScript за пределами браузера. 
- **Deno / Bun** — современные альтернативы Node.js, которые также являются JS Runtime, но с другим набором встроенных API и архитектурой.
