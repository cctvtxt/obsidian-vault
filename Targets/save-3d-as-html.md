# Реальные площадки и пошаговые алгоритмы

## 🏪 МАРКЕТПЛЕЙСЫ (реальные ссылки)

### Blender экосистема

**1. Blender Market**

- 🔗 https://blendermarket.com
- Комиссия: 30-40%
- Выплаты: PayPal

**Алгоритм публикации:**

```
1. Регистрация → blendermarket.com/sign-up
2. Creator Dashboard → "Create Product"
3. Заполнить:
   - Название, описание
   - Цена (мин $5)
   - Скриншоты/видео
   - ZIP файл аддона
   - Документация
4. Модерация: 2-5 дней
5. Публикация
```

**2. Gumroad**

- 🔗 https://gumroad.com
- Комиссия: 10% (или $10/мес для 0% комиссии)
- Работает из России, выплаты через Payoneer/Wise

**Алгоритм:**

```
1. gumroad.com/signup
2. "New Product" → "Digital product"
3. Загрузить ZIP
4. Установить цену (от $1)
5. Настроить "Pay what you want" (опционально)
6. Добавить описание, превью
7. Publish - доступно сразу
8. Получить ссылку: gumroad.com/l/[ваш-код]
```

**3. ArtStation Marketplace**

- 🔗 https://www.artstation.com/marketplace
- Комиссия: 30%
- Фокус: 3D ассеты, инструменты

## 💰 СИСТЕМА ЛИЦЕНЗИРОВАНИЯ

### Простой способ (без защиты):

```python
# Просто проверка файла лицензии
import json

def check_license():
    try:
        with open("license.json", "r") as f:
            license_data = json.load(f)
            if license_data.get("valid"):
                return True
    except:
        pass
    return False
```

### Продвинутый способ:

**Gumroad License API**

- 🔗 https://gumroad.com/api
- Проверка лицензионного ключа онлайн

```python
import requests

def verify_license(license_key):
    url = "https://api.gumroad.com/v2/licenses/verify"
    data = {
        "product_id": "ваш_product_id",
        "license_key": license_key,
        "increment_uses_count": "false"
    }
    response = requests.post(url, data=data)
    return response.json()["success"]
```
