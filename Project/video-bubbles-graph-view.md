### VideoGraph View — как рисовать граф

**Вариант A — Свой граф с нуля (рекомендую)**
- D3.js force-directed (уже есть в Obsidian, можно импортировать)
- Или `cytoscape.js` / `sigma.js` — подключить через npm
- Полный контроль над нодами: md-файлы = обычные круги, видео-ноды = круги с превью
- Связи строятся из wikilinks парсингом `meta.md`

**Вариант B — Хак стандартного графа**
- `app.internalPlugins.plugins['graph'].instance` — доступен
- Можно добавить кастомные ноды через патчинг renderer
- Хрупко, ломается при обновлениях Obsidian

## Технический стек

| Задача           | Технология                                       |
| ---------------- | ------------------------------------------------ |
| Plugin framework | Obsidian Plugin API (TypeScript)                 |
| Граф             | D3.js v7 (force simulation)                      |
| Запись видео     | MediaRecorder API + WebRTC                       |
| Хранение видео   | Blob → ArrayBuffer → vault.adapter.writeBinary() |
| Thumbnail        | Canvas API (захват первого кадра)                |
| Парсинг ссылок   | Obsidian MetadataCache + собственный парсер      |
| Сборка           | esbuild (стандарт для Obsidian плагинов)         |

## Ключевые проблемы и решения

### 🔴 Проблема 1: Размер видео файлов
Vault = папка с файлами, видео может быть гигабайты.

**Решения:**
- Ограничение длины записи (по умолчанию 5 мин)
- Сжатие через `VideoEncoder API` (Chromium 94+, Electron поддерживает)
- Настройка качества записи (360p/720p/1080p)
- Предупреждение при синхронизации (iCloud/Dropbox не любят большие бинарники)

### 🔴 Проблема 2: Производительность графа с видео-превью
Рисовать превью 50+ видео одновременно — тяжело.

**Решения:**
- Lazy loading превью (только для видимых нод)
- Уровни зума: далеко = просто цветной круг, близко = thumbnail, совсем близко = играет видео
- Кэш thumbnail как статичные `.jpg` рядом с `.mp4`

### 🟡 Проблема 3: Синхронизация через git/облако
`.mp4` файлы плохо дружат с git.

**Решения:**
- Документировать что видео-ноты лучше хранить локально или в отдельном vault
- Опциональная интеграция с git-lfs
- Режим "только метаданные" — граф работает даже без видео файлов

### 🟡 Проблема 4: Obsidian mobile
MediaRecorder доступен в мобильных браузерах, но Obsidian mobile = другая среда.

**Решение:** Сначала только desktop, mobile как будущая фича.
