#flashcards/http/3xx

Код HTTP, который указывает на несколько возможных вариантов ресурса. Клиент может выбрать один из них. Сервер предоставляет список вариантов в теле ответа.

300 Multiple Choices

Код HTTP, который означает, что ресурс был окончательно перемещен на новый URL. Важно для SEO, так как передает "вес" страницы новому адресу.
?
301 Moved Permanently
Источник: 300.md
<!--SR:!2026-04-07,3,250-->

Код HTTP, который означает, что ресурс временно находится по другому URL. Клиент должен использовать старый URL для будущих запросов. В отличие от 301, не передает SEO-вес.
?
302 Found
Источник: 300.md
<!--SR:!2026-04-07,3,250-->

Код HTTP, который указывает, что ответ на запрос можно найти по другому URL с помощью метода GET. Используется после POST-запроса, чтобы избежать повторной отправки формы.
?
303 See Other
Источник: 300.md
<!--SR:!2026-04-07,3,250-->

Код HTTP, который означает, что ресурс не изменился с момента последнего запроса. Экономит трафик, так как сервер не отправляет тело ответа.
?
304 Not Modified
Источник: 300.md
<!--SR:!2026-04-06,1,210-->

Код HTTP, который означает, что запрашиваемый ресурс должен быть доступен через прокси. URL прокси указывается в заголовке `Location`.
?
305 Use Proxy
Источник: 300.md
<!--SR:!2026-04-07,2,230-->

Код HTTP, который означает, что он зарезервирован, но больше не используется.
?
306 Unused
Источник: 300.md
<!--SR:!2026-04-07,2,230-->

Код HTTP, который означает, что ресурс временно перемещен, но, в отличие от 302, требует, чтобы клиент не менял метод запроса (например, POST остается POST).
?
307 Temporary Redirect
Источник: 300.md
